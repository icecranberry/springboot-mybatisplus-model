## Introduction

MybatisPlus官方的代码自动生成器示例不是很严谨，我整理了一下网上资料，梳理了一下依赖，稍作修改即可对应你的数据库快速生成清晰的目录及文件，

AutoGenerator的更多参数可以参考[MybatisPlus官方文档](https://mp.baomidou.com/config/generator-config.html#基本配置)。

## How to use
```
1. 修改MpGenerator.java，application.yml，mybatis-plus.properties中标注的数据库地址、表名和项目绝对路径

2. 运行MpGenerator.java

```
